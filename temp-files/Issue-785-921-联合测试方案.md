## 一、当前架构

```
外部调用方                     RAG API 层                   LightRAG
─────────                    ──────────                   ────────
POST /retrieve  ─→ OIDC 认证 → 限流 → 透传 ─→ POST /query
POST /paginated ─→ OIDC 认证 → 限流 → 透传 ─→ POST /documents/paginated
GET /status_counts → OIDC → 限流 → 透传 ─→ GET /documents/status_counts
GET /pipeline_status → OIDC → 限流 → 透传 ─→ GET /documents/pipeline_status
```

四个接口全部纯透传：请求体原样转发，响应原样返回。RAG API 层只负责认证 + 限流，不修改入参和响应。

```
/health + /health/detail    → 健康检查（无认证）
/metrics                    → Prometheus 指标（#785）
```

## 二、环境准备

```bash
BASE="https://<测试域名>"
TOKEN=""       # 通过 OIDC 获取

```

### 获取 token

浏览器访问 `$BASE/api/v1/rag/auth/authorize` → OneID 登录 → 回调返回 JSON → 复制 access_token / refresh_token。

## 十四、一键验收脚本

```bash
#!/bin/bash
set -e
BASE="${1:?Usage: $0 <domain> <token>}"
TOKEN="${2:?Usage: $0 <domain> <token>}"
PASS=0; FAIL=0

check() {
  local code=$1 name=$2
  if [ "$code" = "200" ] || [ "$code" = "500" ]; then
    echo "✅ [$name] HTTP $code"; PASS=$((PASS+1))
  else
    echo "❌ [$name] HTTP $code"; FAIL=$((FAIL+1))
  fi
}

echo "=== 基础 ==="
check "$(curl -s -o /dev/null -w '%{http_code}' $BASE/health)" "health"
check "$(curl -s -o /dev/null -w '%{http_code}' $BASE/metrics)" "metrics"

echo "=== 认证 ==="
check "$(curl -s -o /dev/null -w '%{http_code}' -X POST $BASE/api/v1/rag/retrieve -H 'Content-Type: application/json' -d '{"query":"test"}')" "no-token"
check "$(curl -s -o /dev/null -w '%{http_code}' $BASE/api/v1/rag/documents/status_counts -H 'Authorization: Bearer fake')" "fake-token"

echo "=== 四个接口 ==="
check "$(curl -s -o /dev/null -w '%{http_code}' -X POST $BASE/api/v1/rag/retrieve -H "Authorization: Bearer $TOKEN" -d '{"query":"test"}')" "retrieve"
check "$(curl -s -o /dev/null -w '%{http_code}' $BASE/api/v1/rag/documents/status_counts -H "Authorization: Bearer $TOKEN")" "status_counts"
check "$(curl -s -o /dev/null -w '%{http_code}' $BASE/api/v1/rag/documents/pipeline_status -H "Authorization: Bearer $TOKEN")" "pipeline"
check "$(curl -s -o /dev/null -w '%{http_code}' -X POST $BASE/api/v1/rag/documents/paginated -H "Authorization: Bearer $TOKEN" -d '{"page":1}')" "paginated"

echo "=== 并发 ==="
RESULT=$(seq 20 | xargs -P20 -I{} curl -s -o /dev/null -w '%{http_code}' -X POST $BASE/api/v1/rag/retrieve -H "Authorization: Bearer $TOKEN" -d '{"query":"test"}')
F500=$(echo "$RESULT" | grep -c '500' || echo 0)
if [ "$F500" = "0" ]; then echo "✅ 并发无 500"; else echo "❌ 并发 $F500 个 500"; FAIL=$((FAIL+1)); fi

echo ""
echo "=== 结果: 通过 $PASS / 失败 $FAIL ==="
```
